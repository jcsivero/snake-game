# Snake 🐍

Juego de la serpiente clásico desarrollado en Python con tkinter.

## Requisitos

- Python 3.x (tkinter viene incluido)

## Cómo ejecutar

```bash
python snake.py
```

## Controles

| Tecla | Acción |
|-------|--------|
| `↑` / `W` | Mover arriba |
| `↓` / `S` | Mover abajo |
| `←` / `A` | Mover izquierda |
| `→` / `D` | Mover derecha |
| `R` | Reiniciar |

## Reglas

- Come la comida roja para crecer y sumar **10 puntos**.
- El juego termina si chocas con una pared o contigo mismo.
